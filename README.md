# Just a Minute

A lightweight **web-based focus timer** designed to help users work in short,
pressure-free focus sessions without gamification or distractions.

The project emphasizes **simplicity, usability, and calm interaction design**.

---

## Overview
Just a Minute allows users to set a short focus goal and work uninterrupted,
while providing gentle visual feedback and a clear summary after each session.

All data is stored locally in the browser, ensuring privacy and fast performance.

---

## Features
- Set a focus goal (hours, minutes, seconds)
- Start and stop focus sessions
- Visual progress bar based on the selected goal
- Gentle feedback after each session
- Daily focus summary stored locally in the browser
- No login, no backend, no external services

---

## Example Usage
1. Set a focus goal
2. Click **Start**
3. Work until you stop the timer
4. View session feedback and daily summary

---

## Design Considerations
- Minimal and distraction-free user interface
- Accessibility-aware color choices and layout
- Clear visual feedback without pressure or gamification
- Predictable behavior and simple controls

---

## How to Run
Open `index.html` in any modern web browser.

No installation, build step, or server is required.

---

## Notes
- All data is stored locally using browser storage
- The application works fully offline
- Designed for personal, lightweight use
