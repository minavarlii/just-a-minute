just a minute

A simple web-based focus timer.

Description
just a minute is a lightweight focus timer designed to help users track short focus sessions without pressure or gamification.

Features
• Set a focus goal (hours, minutes, seconds)
• Start and stop focus sessions
• Visual progress bar based on the selected goal
• Gentle feedback after each session
• Daily focus summary stored locally in the browser

Example Usage
1. Set a focus goal
2. Click Start
3. Work until you stop the timer
4. View feedback and daily summary

Technologies
• HTML
• CSS
• JavaScript

How to Run
Open index.html in any modern web browser.

Notes
• No login required
• No backend or server
• Data is stored locally using browser storage
